# Medication (IPS) - International Patient Summary Implementation Guide v2.0.0

## Resource Profile: Medication (IPS) 

 
This profile represents the constraints applied to the Medication resource by the International Patient Summary (IPS) FHIR Implementation Guide, based on FHIR R4. A medication is described in the patient summary as an instance of a Medication resource constrained by this profile. 

**Usages:**

* Use this Profile: [Bundle (IPS)](StructureDefinition-Bundle-uv-ips.md)
* Refer to this Profile: [MedicationRequest (IPS)](StructureDefinition-MedicationRequest-uv-ips.md) and [MedicationStatement (IPS)](StructureDefinition-MedicationStatement-uv-ips.md)
* Examples for this Profile: [Medication/eumfh-39-07-1](Medication-eumfh-39-07-1.md)
* CapabilityStatements using this Profile: [IPS Server Capability Statement](CapabilityStatement-ips-server.md)

You can also check for [usages in the FHIR IG Statistics](https://packages2.fhir.org/xig/hl7.fhir.uv.ips|current/StructureDefinition/Medication-uv-ips)
