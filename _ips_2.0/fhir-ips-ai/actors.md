# Actors - International Patient Summary Implementation Guide v2.0.0

## Actors

### List of IPS Actors

The IPS actors defined for this implementation guide are listed below:

[Consumer (IPS)](ActorDefinition-Consumer.md)

An IPS Consumer actor is a system that receives an IPS document and uses the content of this document.

[Creator (IPS)](ActorDefinition-Creator.md)

An IPS Creator actor is a system which produces/assembles/creates an IPS document.

[Server (IPS)](ActorDefinition-Server.md)

An IPS Server is a FHIR server acting as an [IPS Creator](./ActorDefinition-Creator.md) by providing conformant IPS documents in response to FHIR API requests.
