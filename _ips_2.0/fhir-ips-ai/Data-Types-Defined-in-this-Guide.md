# Data Types Defined in This Guide - International Patient Summary Implementation Guide v2.0.0

## Data Types Defined in This Guide

This guide uses both base [FHIR Data Types](https://hl7.org/fhir/R4/datatypes.html) and datatypes defined below to meet expectations from the ISO 27269 specification.

| | | |
| :--- | :--- | :--- |
| [CodeableConceptIPS](StructureDefinition-CodeableConcept-uv-ips.md) | Codeable Concept (IPS) | This profile represents the constraint applied to the CodeableConcept data type by the International Patient Summary (IPS) FHIR Implementation Guide to use the Coding-uv-ips data type profile. |
| [CodingIPS](StructureDefinition-Coding-uv-ips.md) | Coding with translations | This profile extends the capabilities of the coding data type to support multi-language designations (display). It relies on the Translation extension. |
