# Consumer (IPS) - International Patient Summary Implementation Guide v2.0.0

## ActorDefinition: Consumer (IPS) 

 
An IPS Consumer actor is a system that receives an IPS document and uses the content of this document. 

* **Actor: Consumer**: An IPS Consumer actor is the system that receives an IPS document and uses the content of this document. This includes electronic health records (EHRs), personal health records (PHRs), health information exchanges (HIEs), websites and other applications that display or process an IPS document.
